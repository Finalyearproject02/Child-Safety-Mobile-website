About This Project
These Days parents are worried about their children so they want a complete track of them and monitor them all the time, this is physically not possible so we introduce a Safety Monitoring system which is helpful for monitoring or last geo-location of the child and their activities from anywhere in the world. The major issue of a child missing can be solved with the help of a child tracking system as well as parents who need to keep a track of their every step, this system plays a vital role. This application is divided into 2 Apps, one is for the parent where they can see all the activities of their children and the other is the Child Part, where the child can only see a calculator while the data is been fetched in the background without the child’s knowledge. this is a secure website.
